Project Name: Adaptive Spreading Factor Selection (ASFS) for SX126x LoRa Transceiver

Description: This project provides an implementation of an Adaptive Spreading Factor Selection (ASFS) algorithm for the SX126x LoRa transceiver, utilizing the SX1261MB2BAS shield and STM32 Nucleo L476RG microcontroller. The development environment is set up using Keil uVision 5.
